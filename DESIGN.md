## Design Guidelines — MEGMA HEALTHCARE

### Brand Personality
Professional, clinical, clean, trustworthy. Pharmaceutical-grade credibility. Confident, structured, approved-feeling. Not flashy.

### Color Palette
- Primary: Medical teal/blue (#0A6E7C / #0E7C86) — clinical trust
- Secondary: Deep navy (#14233A) for headers and text
- Accent: Clean green (#2BA84A) for positive/approved/success states
- Surface: White (#FFFFFF) and soft clinical gray (#F4F7F9)
- Borders/dividers: subtle gray (#E2E8ED)
- Warning/alert: amber (#E8A23B); compliance/critical: muted red (#D14343)

### Typography
- Sans-serif system stack (Inter / SF Pro feel). Clean, legible, high-contrast.
- Clear hierarchy: bold navy headings, regular gray body. Generous line spacing for clinical readability.

### Layout & Elevation
- Mobile-first cards with soft rounded corners (8-12px radius)
- Subtle elevation/shadows — never heavy. Flat, organised, calm.
- Generous whitespace. Structured forms with clear labels.
- Bottom tab nav on mobile; sidebar nav on web dashboard.

### Components
- Status pills (approved / pending / dispensed) with color coding
- Data tables and KPI cards for manager dashboard
- Slide presentation viewer (full-screen, swipeable)
- Structured form inputs with validation states
- Avatar + specialty chips for HCP profiles
